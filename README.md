## 简介

本仓库是中国亲戚关系计算器的实现，用于家庭称谓、亲戚称呼、辈分关系的计算。

原始版本为js代码实现，查看更多信息、了解具体算法原理等，请查看原作者仓库 —— [https://github.com/mumuy/relationship](https://github.com/mumuy/relationship)
